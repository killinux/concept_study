# 线性规划（Linear Programming）

## 定义

在**线性约束**下求**线性目标函数**极值。可行域为凸多面体，最优解必在顶点取到。

## 标准形式

```
min  cᵀx
s.t. Ax ≤ b
     x ≥ 0
```

等价变换：max → min取负；≥ → 两边乘-1；等式 → 拆成两个不等式；无界变量 → x = x⁺ - x⁻。

## 核心组件

| 组件 | 说明 |
|------|------|
| 决策变量 x | 待求解的未知量 |
| 目标函数 cᵀx | 要优化的线性函数 |
| 约束条件 Ax ≤ b | 资源/能力限制 |
| 可行域 | 所有约束的交集，凸多面体 |
| 最优解 | 可行域顶点中目标函数最优者 |

## 求解方法

**单纯形法（Simplex）**：从一个顶点出发，沿目标函数改善方向移动到相邻顶点，直到无法改善。最坏指数时间，实际极快。

**内点法（Interior Point）**：从可行域内部穿行趋近最优解。多项式时间，大规模问题有优势。

**对偶单纯形法**：先满足对偶可行性，再逐步恢复原始可行性。适合添加约束后重优化。

## 对偶理论

原问题 min cᵀx, Ax ≥ b, x ≥ 0 的对偶：max bᵀy, Aᵀy ≤ c, y ≥ 0

- **弱对偶**：任何对偶可行解 ≤ 任何原始可行解
- **强对偶**：两者最优值相等（只要有有限最优解）
- **互补松弛**：xᵢ > 0 则对偶第i个约束取等；yⱼ > 0 则原始第j个约束取等
- **影子价格**：对偶变量yⱼ = 第j个资源增加1单位时目标函数的改善量

## 灵敏度分析

| 变化 | 影响 |
|------|------|
| 目标系数 cᵢ 变化 | 最优基不变的允许范围；超出则换基 |
| 右端项 bⱼ 变化 | 最优解按影子价格线性变化（在允许范围内） |
| 新增变量 | 检验数 < 0 则进基，否则当前解仍最优 |
| 新增约束 | 当前解满足则不变，否则用对偶单纯形 |

## 例题：生产计划

工厂生产桌子(x₁)和椅子(x₂)，求最大利润：

```
max  Z = 70x₁ + 50x₂
s.t. 4x₁ + 2x₂ ≤ 120  (木材)
     3x₁ + 5x₂ ≤ 150  (人工)
     x₁, x₂ ≥ 0
```

**求解**：两约束交点 → 4x₁+2x₂=120, 3x₁+5x₂=150 → x₁=150/7≈21.4, x₂=120/7≈17.1

各顶点目标值：O(0,0)=0, (30,0)=2100, (150/7,120/7)≈2357, (0,30)=1500

**最优解**：x₁=150/7, x₂=120/7, Z=16500/7≈2357

**对偶解释**：木材影子价格=10, 人工影子价格=10。多1单位木材利润增10元。

## 经典应用模型

**运输问题**：m个产地向n个销地运货，min总运费，约束为供给量和需求量平衡。

**指派问题**：n人分配n项任务，每人恰做一项，min总成本。本质为0-1规划但LP松弛即为整数解。

**配料问题**：多种原料混合，满足营养/成分约束，min成本。

**网络流**：max流/min费用流均可建模为LP。

## Python求解

```python
from scipy.optimize import linprog

# min -70x1 - 50x2 (取负转为求min)
c = [-70, -50]
A_ub = [[4, 2], [3, 5]]
b_ub = [120, 150]
res = linprog(c, A_ub=A_ub, b_ub=b_ub, bounds=[(0,None),(0,None)])
# res.x ≈ [21.4, 17.1], 最大利润 ≈ 2357
```

```python
# PuLP（更直观的建模语法）
from pulp import *
prob = LpProblem("furniture", LpMaximize)
x1 = LpVariable("tables", lowBound=0)
x2 = LpVariable("chairs", lowBound=0)
prob += 70*x1 + 50*x2
prob += 4*x1 + 2*x2 <= 120
prob += 3*x1 + 5*x2 <= 150
prob.solve(PULP_CBC_CMD(msg=0))
# value(x1)≈21.4, value(x2)≈17.1, value(prob.objective)≈2357
```

## 判断题快速参考

- 可行域无界 → 可能有最优解也可能无界解（取决于目标方向）
- 最优解唯一 ↔ 目标函数等值线与可行域仅交于一个顶点
- 最优解无穷多 ↔ 等值线与某条边重合
- 无可行解 ↔ 约束互相矛盾，可行域为空集
- LP最优解一定在顶点（可能不唯一但至少有一个顶点最优）

## 与其他优化的关系

```
线性规划 ⊂ 凸优化 ⊂ 非线性规划
    ↓
整数规划（变量加整数约束，NP-hard）
    ↓
混合整数规划（部分整数部分连续）
```

LP是整数规划的松弛，也是分支定界法每个节点求解的子问题。掌握LP是学习所有优化方法的基础。
